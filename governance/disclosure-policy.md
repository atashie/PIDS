# Disclosure Policy

*Kept intentionally light (per decision 2026-06-01: "private, minimal process"). Revisit before any public release.*

## Posture
- This repository is **private**.
- Public Git history, preprints, talks, and demos can constitute legal **public disclosure**, which can affect patent rights and journal novelty.

## Minimal rules
1. **Do not make the repository (or any patent-sensitive content) public without an explicit IP review.** When in doubt, ask first.
2. **File before you publish.** For any new invention we intend to protect, ensure the relevant patent filing is in place before public disclosure (papers, posters, marketing, open-sourcing).
3. **Already-published/granted material** is fine to discuss within its disclosed scope (e.g., PIDS application 18/632,186 once published/granted).
4. **Third-party material** is used per its license; provenance recorded in [`../evidence/sources/`](../evidence/sources/).

## When this becomes insufficient
If we add collaborators, pursue additional filings, or consider open-sourcing any component, upgrade this policy (and log the change in [`decision-log.md`](decision-log.md)).
