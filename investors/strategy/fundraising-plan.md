## 2. Strategy in one picture — the milestone-triggered parallel stack

We do **not** run a single linear raise. We run **three funding tracks in parallel**, continuously,
and **gate only the active equity push** behind a milestone trigger:

```
                         ┌─────────────────────────────────────────────┐
   PREPARE NOW  ───────► │  TRACK 1  Equity (SAFE)        — prep now,   │
   (all tracks live)     │           launch on trigger ──► §3            │
                         │  TRACK 2  Non-dilutive R&D + credibility      │
                         │           Lane A: university-partnered        │
                         │           Lane B: no-partner (runs regardless)│
                         │  TRACK 3  Strategic / pilots (validation +    │
                         │           warm-intro engine)                  │
                         └─────────────────────────────────────────────┘
   The tracks reinforce each other: a grant win or a validated pilot raises the
   equity valuation and *is* the trigger that opens Track 1.
```

**Why parallel, not sequential:** grants are slow and congested (SBIR post-lapse), pilots take time,
and equity rewards momentum — running them together means progress on any one track de-risks and
prices the others. **Why milestone-triggered:** a pre-seed left "open" >3 months reads as a failing
process, so we keep Track 1 in *prep* until a milestone lets us run a compressed 6–10 week close.

## 3. Trigger milestones (what converts prep → active equity raise)

Launch the active equity raise when **at least one** decisive de-risking event lands — ideally two:

- **Patent granted** (18/632,186 moves from allowed → issued).
- **A validated field pilot** with quantified, third-party-credible results.
- **Third-party technical sign-off** on the forward-model / physics.
- **A non-dilutive win** (NC IDEA SEED, an SBIR/STTR award, or a fellowship) — elite external
  validation that also raises the SAFE cap.
- **A commercialization-credible co-founder/advisor** joins (retires the team red flag, §8).

> Until a trigger lands, Track 1 work = build materials (§ materials/), build the tiered pipeline
> (§ pipeline/), and run the "not-raising" warm-up tour (§7).

## 4. The funding architecture — three parallel tracks

### Track 1 — Equity (SAFE)  *(prep now, launch on trigger)*
- **Size/instrument:** $500K–$1.5M on a post-money SAFE, cap-only. Use the patent + technical
  de-risking (TRL/MRL story) to justify a higher **cap** rather than accepting more dilution.
- **Targets (tiered, ~50 → ~3 term sheets):**
  - *Local/regional:* IDEA Fund Partners (hardware/spinout seed), RTP Capital, Triangle Angel
    Partners, RTP Angel Fund.
  - *National water/climate specialists (best-fit leads + credibility signal):* Burnt Island, Echo
    River, PureTerra, Emerald; Congruent, Lowercarbon, Earthshot, Clean Energy Ventures.
  - *Hardware/deeptech + agtech:* The Engine, Bee, Bolt, SOSV/HAX; AgFunder, Radicle, ag CVCs.
- **Process:** batch 10–12 first meetings in a tight 1–2 week window; cross-pollinate status to
  manufacture scarcity. Warm intros (esp. via pilot-customer CEOs) ≈ 13× cold-email odds.
- **Local vs. national emphasis — decide post hoc.** No a-priori preference: include *both* local
  (NC IDEA + Triangle angels) and national water specialists in the first batch, then double down on
  whichever channel responds.

### Track 2 — Non-dilutive R&D + credibility  *(runs continuously)*

**Lane B — no university partner required (default, never blocked):**
- **NC IDEA SEED ($50K) / MICRO ($10K)** ✅[C-020/C-021] — directly eligible (NC HQ + ≥1 **full-time**
  NC founder for SEED). **Next cycle: Fall 2026 — opens ~Jul 27, deadline Aug 24, 2026 (5pm EST).**
  Marquee near-term target (SEED deep-dive pending). May apply to only one of MICRO/SEED per cycle.
- **Federal SBIR (not STTR)** — NSF, USDA-NIFA, EPA, DoT all fit our sectors; equity-free. **Treat as
  a 9–18 month play, not a runway bridge** (post-reauthorization congestion).
- **Deeptech fellowships** — **Activate** (~$80–110K/yr + facility support) and **DOE LEEP** (2-yr
  funded, lab access); IP-preserving, climate-mandate fit via water/stormwater framing.
- **Climate accelerator grants** — Elemental Impact ($350K–$1M equity-free), Greentown (up to $50K).

**Lane A — university-partnered (high-value accelerant, §5):**
- **Federal STTR** (+ **One NC** 50% state match **up to $75K** ✅[C-022] — *the prior $60K was the FY2024
  figure*; FY2026 window closes ~Jun 30, target next cycle), **UNC KickStart/OTC** (up to $50K ✅[C-018/C-019],
  gated to a UNC faculty founder + UNC IP — **no alum-direct path**), **NCInnovation** (private companies
  **ineligible** ✅[C-016]; $500M endowment, TRL 3–6; *per-grant range $300K–$1.1M is ⚠️unconfirmed [C-017]*).
  All gated behind a UNC-System research partner. **This Lane now extends well beyond NC — see §5.**

### Track 3 — Strategic / pilots  *(validation + warm-intro engine)*
- **Municipal/utility/sports-field/ag pilots** — design them so a city/county can pay with **public
  water-infrastructure money** (Clean Water State Revolving Fund, EPA stormwater grants, NC water
  boards). Pilots double as the strongest warm-intro source for Track 1.
- **Pilot-driving accelerators:** Joules Cleantech Catalyst (free, no equity, NC utility pilots),
  Techstars WaterTech, Water Innovation Hub; Launch Chapel Hill / Launch-Powered-by-KPMG once an MVP
  is pilot-ready.

## 5. The university-partner keystone — a multi-state portfolio, never a dependency

Attaching **one** university researcher as a scientific co-PI unlocks an analogous stack of state +
university funding (a state applied-research fund + university gap money + federal STTR + a state
SBIR/STTR match) and adds technical credibility. **The keystone mechanism is now primary-verified:**
NCInnovation explicitly **bars private companies** [C-016] and UNC KickStart/OTC gate on a UNC
faculty founder + UNC IP [C-018/C-019] — i.e. an alum cannot reach this money alone; **the professor
relationship is exactly the key.** The same pattern repeats at the four other universities where Arik
has faculty relationships — with sharply different strength (full analysis +
verification: [`../research/2026-06-22-multi-university-academic-funding.md`](../research/2026-06-22-multi-university-academic-funding.md)):

| Partner (state) | What a faculty tie unlocks | Strength |
|---|---|---|
| **U. Alabama** (AL) | **ARDEF** (state R&D, faculty-paired) + **Innovate Alabama Supplemental** = confirmed **50% SBIR/STTR match ≤$250K** [C-015] + **Alabama Water Institute / NOAA CIROH** (best hydrology home) | **Strongest for PIDS** |
| **Virginia Tech + UVA** (VA) | **CCF Higher-Ed Proof-of-Concept** (university-only applicant, $75K–$300K) [C-008] + **UVA LVG ~$10M Seed Fund** + water/stormwater faculty (Sample, Sridhar, Goodall) | **Strong (two schools)** |
| **UNC / NC State** (NC) | **NCInnovation** + **One NC 50% match ≤$75K** [C-022] + **KickStart/OTC** + NC State Stormwater Group (BAE; codifies NC DEQ guidance) | **Home base, verified** |
| **SMU** (TX) | Federal **STTR** + named PI (Smits, MAR/vadose-zone) + Dallas/TWDB recharge pilots — **no state match** (TX has none [C-012]); SMU private → state research funds closed [C-013] | **Weakest funding lever** |

> ⚠️ **State of domicile is now a strategic variable.** Several of these carry in-state residency/HQ
> tests — VA's for-profit CCF path [C-011] and AL's Supplemental Grant (HQ + team in AL, satisfiable
> within 12 months post-award) [C-015]. **Where PIDS incorporates materially affects which doors open** —
> decide deliberately (see §9).

**But it is an *accelerant*, not a gate.** The plan is structured so that **Lane B + Tracks 1 & 3
proceed at full speed without a partner.** If a partner lands, we *add* Lane A on top. We pursue
the strongest-fit partner(s) in parallel — **AL (science + funding) and VA** are the two best analogs
to the UNC opportunity; NC (NC State Stormwater / a UNC faculty inventor) is the home base — with a
**founder-signaled fallback: no pegged deadline** — Arik calls the timeout. If/when he signals the
partner track has stalled, we proceed on Lane B alone and revisit at seed.

## 6. Instrument & terms guardrails

- Post-money SAFE, **cap-only, no discount.** Avoid stacking many SAFEs blindly — **track cumulative
  promised ownership** (the post-money "dilution trap": $1.5M across three $500K SAFEs at a $10M cap
  ≈ ~15% before any priced round).
- A value-adding partner at a fair cap beats a higher cap from an absent check.
- Keep a **clean cap table incl. all SAFEs** in the data room from day one (missing it is a red flag).
- Non-dilutive ≠ free: SBIR is R&D-scoped, effort-heavy (NSF Phase I ≈ 80–120 founder-hours), and
  slow — don't let grant-chasing displace pilots/early revenue.

## 7. Sequencing & timeline

**Phase 0 — Prepare now (current):** finalize this plan; build the narrative + deck-outline +
one-pager + data-room index (§ materials/); build the tiered ~50-name pipeline (§ pipeline/); stand
up the grants tracker (§ grants/); open the university-partner conversation; begin the **"not-raising"

**Phase 1 — Near-term non-dilutive + pilots (parallel, now → ~6 mo):** **NC IDEA (Fall 2026 window —
application opens ~Jul 27, deadline Aug 24, 2026)**, Joules, Activate/LEEP next cycle, scope a
municipal pilot, file SBIR when the relevant agency reopens. Open the strongest academic-partner
conversations (AL Water Institute / VA water faculty / NC State BAE) in parallel.

**Phase 2 — Trigger & compressed equity raise:** when a §3 milestone lands, run the active raise in a
tight **6–10 week** window (batch meetings → diligence → close), leveraging warm relationships from
Phase 0 and validation from Phases 1.

**Phase 3 — Post-raise / seed bridge (out of scope for now):** FOAK/deployment financing (debt +
project finance + offtake) deferred to post-traction.

## 8. Risks & guardrails

- **Team-credibility red flag (highest priority).** A solo science-heavy founder is a known pre-seed
  red flag → recruit a commercialization co-founder/advisor *before or during* the raise; it's also a
  trigger milestone (§3) and a Launch Chapel Hill prerequisite.
- **SBIR congestion** — reauthorized Apr 2026 but backlogged; do not budget runway on it.
- **Grant-chasing displacement** — grants are slower/more conditional than customer cash; cap the
  founder-hours spent chasing them.
- **Jargon risk** — the Richards/FEM story must be translated for generalist investors or it lands in
  the reject pile.
- **Defense/dual-use distraction** — DIU/AFWERX is credible (drone mapping + terrain/water modeling)
  but pulls roadmap toward defense; hold as optional upside only.
