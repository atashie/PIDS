# Installation Methods

## 1. Purpose
The field techniques that build PIDS: trenching the conveyance-channel network, augering the vertical infiltration tunnels, and placing/compacting materials (including tamping the bentonite base). The execution science that determines whether the as-built system matches the designed configuration.

## 2. Status
- Maturity: concept
- Last reviewed: 2026-06-01

## 3. Scientific basis
- Channels: shallow trenching (~1–2 ft).
- Tunnels: augering to ~6 ft (or to bedrock/saprolite).
- Barriers: layer placement + compaction ("tamper": a tool to compress/pack material).

## 4. Dependencies (outgoing)
- None (execution layer). Realizes the geometry designed by [`../../infiltration-runoff-model/`](../../infiltration-runoff-model/).

## 5. Interfaces
- Inputs: design geometry (spacing, depth, placements).
- Outputs: as-built geometry + deviations → model validation.
- Assumptions: equipment can achieve specified depths/placements at target sites.
