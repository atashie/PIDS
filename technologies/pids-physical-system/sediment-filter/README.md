# Sediment Filter

## 1. Purpose
The layered structure within a conveyance channel — a sand top layer over a clay base — that filters sediment from incoming surface water while (where the clay base is present) limiting downward percolation. In the granted claim language this is the channel's "top layer and water-impermeable bottom layer."

## 2. Status
- Maturity: concept
- Last reviewed: 2026-06-01

## 3. Scientific basis
- Sand top layer: sediment size ~0.5–2 mm (coarse, optionally rounded).
- Clay base: compacted bentonite (see [`../impermeable-barriers/`](../impermeable-barriers/)).

## 4. Dependencies (outgoing)
- [`../conveyance-channels/`](../conveyance-channels/) — the host structure.
- [`../materials/`](../materials/) — sand and clay.

## 5. Interfaces
- Inputs: sand grading, clay spec.
- Outputs: filtered water; reduced clogging of the conveyance fill.
- Assumptions: grain sizes chosen to filter target sediment without excessive clogging.
